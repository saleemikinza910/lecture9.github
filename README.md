 🏡 VistaPrime – Advanced Real Estate Platform

VistaPrime is a **modern, fully interactive real estate web application** built using HTML, CSS, Bootstrap 5, and Vanilla JavaScript. It simulates a real-world property marketplace with advanced features like filtering, comparison, mortgage tools, maps, and user personalization.

---

 🌟 Live Features Overview

🏠 1. Property Listings System
- 16 premium estate listings
- Real images, prices, locations, and specifications
- Property status tracking (Owned / Sold / Available)
- Responsive grid layout for all devices

---

🔍 2. Smart Filtering & Sorting
- Filter by:
  - City
  - Marla size (5, 10, 20 Marla, 1 Kanal)
  - Maximum price range slider
- Sorting options:
  - Price (Low → High / High → Low)
  - Name (A → Z)
- Instant real-time filtering without page reload

---

 ❤️ 3. Wishlist System
- Add properties to wishlist
- Remove items anytime
- Persistent storage using localStorage
- Live wishlist counter

---

 ⚖️ 4. Property Comparison Tool
- Compare up to 3 properties simultaneously
- Side-by-side comparison table
- Includes:
  - Price
  - Location
  - Marla size
  - Monthly mortgage estimate

---

 💰 5. Mortgage Calculator
- Monthly installment calculator
- Includes:
  - Loan interest (5%)
  - Tax & insurance estimation
- Real-time amortization table (first 12 months)
- Fully interactive UI

---

 🏦 6. Mortgage Pre-Approval System
- Calculates maximum affordable property price
- Based on:
  - Annual income
  - Monthly debts
  - Down payment
- Uses standard 43% debt-to-income rule
- Instant eligibility feedback

---

📊 7. Rent vs Buy Analysis
- Compares renting vs buying cost
- Shows monthly savings or loss
- Helps users make financial decisions

---

 🗺️ 8. Interactive Map Integration (Leaflet.js)
- Displays all properties on live map
- Property markers with details popup
- User geolocation support
- “Find nearest property” feature

---

 👤 9. Authentication System (Demo)
- Login / Signup modal
- Demo credentials system
- User session stored in localStorage
- Personalized navbar experience

---

📁 10. Portfolio Management
- Track owned and sold properties
- Portfolio statistics:
  - Total properties
  - Sold properties
  - Available properties
- Reset portfolio option

---

 👁️ 11. Recently Viewed System
- Tracks last viewed properties automatically
- Stores up to 5 recent items
- Delete individual history entries

---

 📩 12. Inquiry System
- Send inquiries from property modal
- Stores messages in localStorage
- Includes property + user details

---

 🌙 13. Dark Mode Support
- Toggle between light & dark themes
- Persistent theme saving
- Smooth UI transitions

---

 🛠️ Tech Stack

- HTML5
- CSS3
- Bootstrap 5
- JavaScript (ES6)
- Leaflet.js (Maps API)
- LocalStorage (Data persistence)
